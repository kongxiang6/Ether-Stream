Ether Stream 中文使用说明

一、程序是做什么的

这套程序分为两部分：

1. 发射端
作用：抓取主机屏幕画面，压缩成 JPEG 后，通过原始以太网帧发送出去。

2. 接收端
作用：接收发射端发来的原始以太网分片，重组后在界面实时预览，并通过 UDP 转发给本机或其它程序。

补充：
接收端转发出去的 UDP 数据，不是标准视频流。
它的格式是“每个 UDP 包里放一张完整 JPEG”。
如果你使用第三方 Preview 程序，必须确认它支持这种格式，否则就会出现端口有数据、但窗口没画面的情况。


二、使用前准备

1. 两台电脑都需要安装 Npcap。
2. 建议两边都用“管理员身份运行”。
3. 两台电脑最好网线直连，或者接在同一个交换机上。


三、发布包里的文件

1. 文件夹版\发射端\发射端.exe
发射端程序（文件夹版）

2. 文件夹版\接收端\接收端.exe
接收端程序（文件夹版）

3. 单文件版\发射端-单文件版.exe
发射端单文件版

4. 单文件版\接收端-单文件版.exe
接收端单文件版

Npcap 安装包

中文操作说明

项目说明文档

补充：
GitHub 仓库里建议只放源码。
成品 exe、文件夹版、单文件版压缩包、npcap 安装包，建议放到 GitHub 的“Releases / 发布”里。
仓库本身只保留源码、说明文档和打包脚本即可。


四、正确启动顺序

建议按下面顺序操作：

1. 先开接收端
2. 选择监听网卡，点击“开始接收”
3. 再开发射端
4. 选择网卡，填写目标 MAC
5. 点击“开始发射”


五、接收端怎么填

1. 监听网卡
直接优先选带【推荐】的那张网卡。

2. UDP 转发地址
默认先用：
127.0.0.1:4455

说明：
当前版本接收端已经内置实时预览窗口。
就算你没有额外的 UDP 播放程序，也可以直接在接收端界面看到画面。
这个输入框格式就是：
IP:端口

例如：
192.168.6.1:1122

选择监听网卡后，程序会自动把这个地址带成当前网卡的本机 IP，并保留端口号。

3. 队列长度
默认：
4096

4. 拼包超时(秒)
默认：
0.25

一般这三个参数保持默认就可以。


六、发射端怎么填

1. 网卡接口
直接优先选带【推荐】的那张网卡。

2. 目标 MAC
这是最重要的参数，必须填写。
这里填“接收端网卡”的 MAC 地址。

示例：
80:fa:5b:60:12:34

3. 采集帧率
默认：
60

也可以提高到：
120
150
300

说明：
帧率越高，对 CPU、JPEG 编码、网卡发送和接收端解码压力越大。
不是数值越高就一定越流畅，建议先用 60 测试，再逐步往上调。

4. JPEG 质量
默认：
80

5. 单帧负载预算
默认：
1400

6. 采集区域
这个版本已经改成“屏幕中心正方形区域”。

填写方法：

如果填：
320

表示抓屏幕正中间的 320x320 区域。

如果填：
480

表示抓屏幕正中间的 480x480 区域。

如果留空：
表示抓整个屏幕。

补充：
如果发射端程序窗口本身就放在屏幕正中间，
那抓到的中心区域里也会包含这个程序窗口。


七、推荐测试参数

接收端：

监听网卡：选【推荐】
UDP 转发地址：默认自动带出当前监听网卡 IP，并保留端口
队列长度：4096
拼包超时：0.25

发射端：

网卡接口：选【推荐】
目标 MAC：填写接收端 MAC
采集帧率：60
JPEG 质量：80
单帧负载预算：1400
采集区域：320


八、怎么查接收端的 MAC 地址

在接收端电脑打开命令提示符，输入：

ipconfig /all

找到对应网卡的“物理地址”。

例如看到：
80-FA-5B-60-12-34

在发射端里要写成：
80:fa:5b:60:12:34


九、常见问题

1. 提示 invalid MAC address: ''
原因：目标 MAC 没填。
处理：把接收端网卡的 MAC 地址填到发射端“目标 MAC”里。

2. 不知道选哪张网卡
处理：优先选界面里带【推荐】的那一项。

3. 检测不到网卡
原因：通常没安装 Npcap，或者程序不是管理员运行。
处理：先安装 Npcap，再用管理员身份运行。

4. 只复制 exe 到别的电脑打不开
原因：这是目录版打包，依赖在 _internal 文件夹里。
处理：必须整个文件夹一起复制，不能只复制 exe。

5. 单文件版为什么启动更慢
原因：单文件 exe 启动时会先把运行依赖解压到系统临时目录。
处理：这是正常现象。想要启动更快，就用文件夹版。

6. 有画面但卡顿
可以尝试：
把采集帧率从 60 降到 30
把 JPEG 质量从 85 降到 75 或 80
把采集区域从 480 改成 320


十、其它电脑怎么用

1. 先安装 Npcap
2. 解压整个发布包
3. 保持目录结构不变
4. 不要删除 _internal 文件夹
5. 再运行：
文件夹版\发射端\发射端.exe
或：
文件夹版\接收端\接收端.exe

如果你想直接发一个 exe 给别人：

1. 先让对方安装 Npcap
2. 使用 单文件版 目录里的单文件版
3. 直接发：
发射端-单文件版.exe
或：
接收端-单文件版.exe
